Algorithm 5.1.14.

Suppose \(V' = \{\bfv_1, \ldots, \bfv_n\}\) is a set of vectors in a vector space \(\ff^m\text{.}\) In order to determine whether or not \(V'\) is linearly independent, we follow these steps.
  1. Form the matrix \(A = [\bfv_1 \cdots \bfv_n]\) and row reduce it to REF. Call this matrix \(B\text{.}\)
  2. The set \(V'\) is linearly independent if and only if \(B\) has a pivot in every column.
