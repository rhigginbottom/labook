Theorem 5.4.10. The Rank-Nullity Theorem.

If \(A \in M_{m,n}(\ff)\text{,}\) then
\begin{equation*} \rank(A) + \dim(\nll(A)) = n\text{.} \end{equation*}
If \(T \in L(V,W)\text{,}\) then
\begin{equation*} \rank(T) + \dim(\ker(T)) = \dim(V)\text{.} \end{equation*}

Proof.

We will prove the result for matrices. The proof for linear transformations is a bit more technical. (The reader should note that the result for linear transformations implies the result for matrices!)
If \(A \in M_{m,n}(\ff)\text{,}\) let \(B\) be the RREF of \(A\text{.}\) Then \(\rank(A)\) is the number of pivot columns in \(B\text{.}\) Further, \(\dim(\nll(A))\) is the number of non-pivot columns in \(B\text{.}\) (See ExerciseÂ 5.3.4.8.) Since each of the \(n\) columns of \(B\) must be either a pivot or a non-pivot column, and since \(A\) and \(B\) have the same number of columns, this proves the theorem.
