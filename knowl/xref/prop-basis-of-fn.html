Proposition 5.2.9.

The set \(\{\bfv_1, \ldots, \bfv_m \}\) is a basis for \(\ff^n\) if and only if the RREF of the matrix \([\bfv_1 \cdots \bfv_m]\) is \(I_n\text{.}\)

Proof.

From Theorem 3.4.14 we know that the set \(V' = \{\bfv_1, \ldots, \bfv_m \}\) spans \(\ff^m\) if and only if the RREF of \(A=[\bfv_1 \cdots \bfv_m]\) has a pivot in every row. Additionally, Algorithm 5.1.14 tells us that \(V'\) is linearly independent if and only if the RREF of \(A\) has a pivot in each column. The only way a matrix in RREF can have a pivot in every row and every column is if that RREF is the identity matrix.
