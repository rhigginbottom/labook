Proposition 5.5.6.

Let \(V\) be a vector space of dimension \(m\) over \(\ff\text{.}\) Since the coordinate mapping \(C_{\mcb}:V \to \ff^m\) is an isomorphism, then the following statements are true.
  1. A set of vectors \(\{\bfv_1, \ldots, \bfv_n \}\) in \(V\) is linearly independent if and only if the set of coordinate vectors \(\{C_{\mcb}(\bfv_1), \ldots, C_{\mcb}(\bfv_n) \}\) is linearly independent in \(\ff^m\text{.}\)

    🔗
  2. A set of vectors \(\{\bfv_1, \ldots, \bfv_n \}\) spans \(V\) if and only if the set of coordinate vectors \(\{C_{\mcb}(\bfv_1), \ldots, C_{\mcb}(\bfv_n) \}\) spans \(\ff^m\text{.}\)
