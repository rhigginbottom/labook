Theorem 5.3.2.

Suppose that, for every \(n \ge 1\text{,}\) a vector space \(V\) contains a linearly independent subset of size \(n\text{.}\) Then \(V\) is infinite-dimensional.

Proof.

We will prove the contrapositive. If \(V\) is finite-dimensional, then there exists a set \(V' = \{\bfv_1, \ldots, \bfv_m\}\) such that \(V = \spn(V')\text{.}\) By LemmaÂ 5.3.1, for any \(n > m\text{,}\) \(V\) cannot contain a linearly independent set of \(n\) vectors. This completes the proof of the contrapositive.
