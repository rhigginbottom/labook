Theorem 5.4.13. The Invertible Matrix Theorem.

Let \(A \in M_n(\ff)\text{.}\) Then the following statements are equivalent.
  1. The matrix \(A\) is invertible.
  2. The matrix \(A\) is row equivalent to \(I_n\text{.}\)
  3. The matrix \(A\) has \(n\) pivots.
  4. The equation \(A\bfx = \bfo\) has only the trivial solution.
  5. The columns of \(A\) form a linearly independent set.
  6. If \(T_A:\ff^n \to \ff^n\) is multiplication by \(A\text{,}\) then \(T\) is injective.
  7. The equation \(A\bfx = \bfb\) has a solution for all \(\bfb \in \ff^n\text{.}\)
  8. The columns of \(A\) span \(\rr^n\text{.}\)
  9. The linear transformation \(T_A\) is surjective.
  10. There is an \(n\times n\) matrix \(B\) such that \(AB = BA = I_n\text{.}\)
  11. The matrix \(A^T\) is invertible.
  12. We have \(\det(A) \neq 0\text{.}\)
  13. The columns of \(A\) form a basis for \(\ff^n\text{.}\)
  14. We have \(\col(A) = \ff^n\text{.}\)
  15. We have \(\rank(A) = n\text{.}\)
  16. We have \(\nll(A) = \{\bfo\}\text{.}\)
  17. We have \(\dim(\nll(A)) = 0\text{.}\)
