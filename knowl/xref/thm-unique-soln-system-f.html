Theorem 2.2.3.

Suppose that \(A\) is the augmented matrix corresponding to a consistent \(m\times n\) linear system over a field \(\ff\text{,}\) and suppose that \(A\) is in reduced row-echelon form. Then the system has a unique solution if and only if there is a pivot in each of the first \(n\) columns of \(A\text{.}\)
