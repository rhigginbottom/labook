Example 3.4.5.

Let \(A\) be the matrix
\begin{equation*} A = \begin{bmatrix} 3 \amp 1 \amp -3 \\ -1 \amp 0 \amp 2 \\ 2 \amp -2 \amp -10 \end{bmatrix}\text{.} \end{equation*}
We can find \(\nll(A)\) by row reducing the matrix \([A \mid \bfo]\text{.}\) Here is the RREF:
\begin{equation*} \left[ \begin{array}{@{}ccc|c@{}} 1 \amp 0 \amp -2 \amp 0 \\ 0 \amp 1 \amp 3 \amp 0 \\ 0 \amp 0 \amp 0 \amp 0 \end{array}\right]\text{.} \end{equation*}
From this we see that \(x_1 = 2x_3\text{,}\) \(x_2=-3x_3\text{,}\) and \(x_3\) is free. In other words, any vector \(\bfx\) in \(\nll(A)\) looks like
\begin{equation*} \bfx = \begin{bmatrix} 2x_3 \\ -3x_3 \\ x_3 \end{bmatrix} = x_3\begin{bmatrix} 2 \\ -3 \\ 1 \end{bmatrix}\text{.} \end{equation*}
So we have
\begin{equation*} \nll(A) = \spn\{\bfv\} \end{equation*}
if
\begin{equation*} \bfv = \begin{bmatrix} 2 \\ -3 \\ 1 \end{bmatrix}\text{.} \end{equation*}
