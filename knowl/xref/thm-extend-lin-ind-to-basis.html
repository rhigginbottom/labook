Theorem 5.3.16.

Suppose that \(V\) is a finite-dimensional vector space and that \(V'\) is a linearly independent set of vectors in \(V\text{.}\) Then there is a basis of \(V\) which contains \(V'\text{.}\)

Proof.

Let \(V' = \{\bfv_1, \ldots \bfv_n \}\) be a linearly independent set of vectors in \(V\text{.}\) If \(V = \spn(V')\text{,}\) then \(V'\) is a basis and we are done. If \(V \neq \spn(V')\text{,}\) then there exists some vector \(\bfv_{n+1} \in V - \spn(V')\text{.}\) By the Linear Dependence Lemma (TheoremÂ 5.1.19), the set \(V_1 = V' \cup \{\bfv_{n+1} \}\) is linearly independent.
