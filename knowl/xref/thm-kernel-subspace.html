Theorem 3.4.2.

Let \(T:V\to W\) be a linear transformation. Then \(\kerr(T)\) is a subspace of \(V\text{.}\)

Proof.

Since \(T(\bfo_V) = \bfo_W\text{,}\) we have \(\bfo_V \in \kerr(T)\text{.}\) (The fact that \(T(\bfo_V)=\bfo_W\) is found in ExerciseÂ 3.1.6.10.)
Let \(\bfv_1,\bfv_2 \in \kerr(T)\text{.}\) Using the additive property of \(T\) and the fact that these vectors are in the kernel, we have
\begin{equation*} T(\bfv_1 + \bfv_2) = T(\bfv_1) + T(\bfv_2) = \bfo_W + \bfo_W = \bfo_W\text{.} \end{equation*}
Finally, we let \(\bfv \in \kerr(T)\) and \(k \in \ff\text{.}\) Then we have
\begin{equation*} T(k\bfv) = kT(\bfv) = k\bfo_W = \bfo_W\text{.} \end{equation*}
This calculation used the fact that \(\bfv\) was assumed to be in the kernel and the scalar multiplication property of \(T\text{.}\)
