Example 7.2.6.

Consider the following inner product on the space \(P_1\text{:}\)
\begin{equation*} \ip{p,q} = p(0)q(0) + p(1)q(1)\text{.} \end{equation*}
The set \(\mcb = \{t, 1-t \}\) is an orthonormal basis for \(P_1\text{.}\) We first verify that this is an orthogonal set:
\begin{equation*} \ip{t, 1-t} = (0)(1) + (1)(0) = 0\text{.} \end{equation*}
We can also see that each of these are unit vectors:
\begin{align*} \ip{t,t} \amp = 0^2 + 1^2 = 1\\ \ip{1-t, 1-t} \amp = 1^2 + 0^2 = 1\text{.} \end{align*}
Thus, \(\mcb\) is an orthonormal basis for \(P_1\) with this inner product.
