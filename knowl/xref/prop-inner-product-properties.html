Proposition 7.1.17.

Suppose that \(V\) is an inner product space. Then the following statements are all true.
  1. For each \(\bfu, \bfv, \bfw \in V\text{,}\) \(\lla \bfu, \bfv + \bfw \rra = \lla \bfu, \bfv \rra + \lla \bfu, \bfw \rra\text{.}\)
    🔗
    🔗
  2. For each \(\bfu, \bfv \in V\) and each \(c \in \ff\text{,}\) \(\lla \bfu, c\bfv \rra = \overline{c} \lla \bfu, \bfv \rra\text{.}\)
    🔗
    🔗
  3. For each \(\bfu \in V\text{,}\) \(\lla \bfo, \bfu \rra = \lla \bfu, \bfo \rra = 0\text{.}\)
    🔗
    🔗
  4. If \(\bfu \in V\) and \(\lla \bfu, \bfv \rra = 0\) for every \(\bfv \in V\text{,}\) then \(\bfu = \bfo\text{.}\)
    🔗
    🔗
🔗
🔗

Proof.
We will prove the second property and leave the others for the exercises. Using the first and third axioms from the definition of the inner product, we have
\begin{equation*} \lla \bfu, c\bfv \rra = \overline{\lla c\bfv, \bfu \rra} = \overline{c\lla \bfv, \bfu \rra} = \overline{c} \overline{\lla \bfv, \bfu \rra } = \overline{c} \lla \bfu, \bfv \rra\text{.} \end{equation*}
