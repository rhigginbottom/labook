Fact 1.4.10.

For all vectors \(\bfu\text{,}\) \(\bfv\text{,}\) and \(\bfw\) in \(\rr^n\text{,}\) and for all real numbers \(c\) and \(d\text{,}\) the following properties hold.
  1. \(\displaystyle \bfu + \bfv = \bfv + \bfu\)
    🔗
  2. \(\displaystyle (\bfu + \bfv) + \bfw = \bfu + (\bfv+\bfw)\)
    🔗
  3. \(\displaystyle \bfu + \mathbf{0} = \mathbf{0}+\bfu = \bfu\)
    🔗
  4. \(\displaystyle \bfu + (-\bfu) = (-\bfu) + \bfu = \mathbf{0}\)
    🔗
  5. \(\displaystyle c(\bfu+\bfv) = c\bfu + c\bfv\)
    🔗
  6. \(\displaystyle (c+d)\bfu = c\bfu + d\bfu\)
    🔗
  7. \(\displaystyle c(d\bfu) = (cd)\bfu\)
    🔗
  8. \(\displaystyle 1\bfu = \bfu\)
    🔗
