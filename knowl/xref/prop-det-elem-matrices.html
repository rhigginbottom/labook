Proposition 4.2.8.

Let \(E \in M_n(\ff)\) be an elementary matrix.
  1. If \(E\) performs the switch elementary row operation, then \(\det(E)=-1\text{.}\)
  2. If \(E\) performs the scale elementary row operation, and if the scaling is by \(c \in \ff\text{,}\) then \(\det(E) = c\text{.}\)
  3. If \(E\) performs the replace elementary row operation, then \(\det(E) = 1\text{.}\)

Proof.

Every elementary matrix in \(M_n(\ff)\) is the result of performing a single elementary row operation on \(I_n\text{.}\) We have theorems in this section which tell us how these elementary row operations affect the determinant of a matrix, and since from Lemma 4.2.7 we know that \(\det(I_n)=1\text{,}\) we will be able to arrive at our result.
If \(E\) performs a switch row operation, then by Theorem 4.2.3 we have \(\det(E) = \det(EI_n) = -\det(I_n)=-1\text{.}\)
If \(E\) scales one row of a matrix by a non-zero \(c \in \ff\text{,}\) then by Theorem 4.2.4 we have \(\det(E) = \det(EI_n) = c\det(I_n) = c\text{.}\)
Finally, if \(E\) performs a replace row operation, then by Theorem 4.2.5 we have \(\det(E) = \det(EI_n) = \det(I_n) = 1\text{,}\) which completes the proof.
